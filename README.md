# 陵辰 · 世界线观测终端

一个零依赖、可直接部署的《命运石之门》氛围个人观测终端。站名与作者统一为“陵辰”，不公开邮箱和所在城市。

## 本地查看

直接双击根目录的 `index.html` 即可浏览。也可以在本目录启动任意静态文件服务器后访问首页。

## 设计与功能

- 私人桌面式首页：原创复古时间实验室插画与非对称组件布局
- CRT 磷光时钟、自动月历和世界线时段问候
- 随机世界线信号、音乐播放动效、本地点赞和明暗模式
- 空公开档案、研究员档案与实习观测日志终端
- 每日实习日志新增、编辑、删除、检索、TXT/Markdown 读取，以及 JSON 导入导出备份
- 桌面、平板和手机三档响应式布局
- 泛黄实验档案、扫描线、电路与橙红警示色组成的复古科幻主题
- 本地原生播放器：茶理理《星愿 off vocal》
- RSS、站点地图、robots.txt 与 404 页面

## 内容说明

当前没有公开文章，公开档案、RSS 和站点地图中的旧文章记录均已清空。实习观测日志使用浏览器 `localStorage` 保存，只存在于当前浏览器；请定期通过页面中的“导出备份”保存 JSON 文件。

## 发布

整个 `lingchen-blog` 文件夹可以直接部署到 GitHub Pages、Cloudflare Pages、Netlify 或 Vercel，不需要构建命令。当前公开地址为 `https://lingchen000.github.io/-/`。
